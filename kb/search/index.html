Using SGDG Search

About

SGDG Search is a great way for you to find projects, people, events, and more SGDG-related things. We've included a bunch of functionality into this search feature, but we hope to add more as needed.

Searching

You can filter the results by type using the checkboxes below the search bar. Only results for types that are checked will appear.

You can search for projects by name, project type, and description.

You can search for people by name, role, description, and interests/skills.

You can include more than one search term by separating terms with commas (,) in the search bar. Results are produced based on whether they match any of the terms ('OR'). For example, if you search for people by typing 'programming,game design', any person that has at least one of those interests/skills will appear in the results.

You can type 'all' to display literally everything. For example, if you want a list of all SGDG members, check only the 'People' box and type 'all'.
